# Detection and Prediction of Alzheimer’s Disease

## Project Overview
This project uses machine learning techniques to detect and predict Alzheimer’s disease based on medical and cognitive data. The aim is to support early identification of Alzheimer’s risk using data-driven analysis.

## Problem Statement
Alzheimer’s disease is difficult to detect at an early stage. This project focuses on building a classification model that can help predict disease presence or stage using available patient-related features.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Machine Learning Classification

## Key Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis
- Feature selection
- Model training
- Model evaluation

## Model Evaluation
The model was evaluated using:
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

## Outcome
The project demonstrates how machine learning can support healthcare-related prediction tasks and assist in early disease detection.
